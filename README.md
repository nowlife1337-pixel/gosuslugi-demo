![ГосFake](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-active-success)

## 📋 Описание

**ГосFake** — это развлекательный пародийный веб-сервис, имитирующий интерфейс приложения «Госуслуги». Сервис позволяет создавать реалистичные демонстрационные версии цифровых документов:

- 🛂 Паспорт РФ
- 🟢 СНИЛС
- 🔢 ИНН
- 🏥 Полис ОМС
- 🚗 Водительское удостоверение

## ⚠️ Важно

**Данный сервис является развлекательным пародийным проектом** и не имеет никакого отношения к порталу «Госуслуги» или каким-либо государственным органам Российской Федерации.

- Все данные являются вымышленными и вводятся самим пользователем
- Сервис не создаёт, не подделывает и не распространяет реальные документы
- Результаты работы сервиса не имеют юридической силы
- Использование в мошеннических целях строго запрещено

## ✨ Возможности

### Для пользователей

- ⚡ **Быстрый вход** — генерация ключа доступа за 3 секунды
- 📄 **Реалистичные документы** — точная копия интерфейса Госуслуг
- 📱 **Адаптивный дизайн** — работает на всех устройствах
- 🔒 **Локальное хранение** — все данные хранятся только в вашем браузере
- 🎨 **Тёмная тема** — переключение между светлой и тёмной темой
- 📲 **QR-коды** — генерация QR-кодов для каждого документа

### Для администраторов

- 🔧 **Редактирование документов** — изменение данных всех документов
- 📊 **Логи входов** — просмотр последних 50 входов пользователей
- 🔐 **Защищённый доступ** — вход по паролю (долгое нажатие 3 сек на аватар)

## 🚀 Быстрый старт

### Онлайн версия

Просто откройте: [https://nowlife1337-pixel.github.io/gosuslugi-demo/](https://nowlife1337-pixel.github.io/gosuslugi-demo/)

### Локальная установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/nowlife1337-pixel/gosuslugi-demo.git
cd gosuslugi-demo
```

2. Откройте `index.html` в браузере:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

Или запустите локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

Затем откройте: `http://localhost:8000`

## 📖 Использование

### Получение ключа доступа

1. Нажмите кнопку **"Начать бесплатно"**
2. Система сгенерирует ключ формата `GOS-XXXXXXXXXXXX`
3. Сохраните ключ — он действителен 24 часа
4. Автоматический вход в приложение

### Вход с существующим ключом

1. Нажмите кнопку **"Войти"**
2. Введите ваш ключ доступа
3. Нажмите **"Войти"**

### Просмотр документов

1. На главном экране выберите документ
2. Нажмите **"Показать все данные"** для полной информации
3. Нажмите **"QR-код"** для генерации QR-кода

### Доступ к админ-панели

1. Удерживайте аватар пользователя **3 секунды**
2. Введите пароль: `admin123`
3. Редактируйте документы или просматривайте логи
